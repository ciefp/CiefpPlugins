# CiefpVideoPlayer v1.3

**CiefpVideoPlayer** — это продвинутый видеоплеер для устройств Enigma2 с полным контролем воспроизведения.

## 🎬 Основные функции

### 📺 Расширенный видеоплеер
- **Полное управление воспроизведением** – Воспроизведение, Пауза, Стоп, Перемотка назад, Перемотка вперед
- **Запоминание позиции** – продолжение просмотра с того места, где вы остановились
- **HTTP-потоки** – пауза и возобновление просмотра онлайн-контента
- **Субтитры** – поддержка встроенных и внешних субтитров

### 📁 Поддерживаемые источники
- **Локальные файлы** – HDD, USB, CF-карты
- **Сетевые ресурсы** – SMB/CIFS, NFS
- **Онлайн-потоки** – плейлисты M3U и Enigma2 .tv
- **Открытый каталог** – просмотр и воспроизведение фильмов/сериалов с HTTP-серверов

### 🎭 Интеграция с TMDB
- **Фильмы** – подробная информация, рейтинги, актеры (желтая кнопка)
- **Сериалы** – информация о сезонах, Эпизоды, актеры (синяя кнопка)
- **Постеры** – автоматическое кэширование для более быстрого просмотра
- **Рейтинги IMDb** – через API OMDb (опционально)

### 💾 Кэширование
- Постеры и метаданные кэшируются для более быстрой загрузки
- Регулируемый размер кэша
- Опция автоматической очистки при выходе

## 🎮 Элементы управления MoviePlayer

| Кнопка | Функция |

|------|------|

| **Воспроизведение/Пауза** | Пауза и возобновление |

| **Стоп** | Остановить воспроизведение |

| **◀ ▶** | Перемотка назад/вперед |

| **◀◀ ▶▶** | Сканирование (перемотка вперед) |

| **Информация** | Показать информацию о видео |

| **Аудио** | Выбрать звуковую дорожку |

| **Субтитры** | Включить/выключить субтитры |

**Выход** | Выйти из плеера (запоминает позицию!) |

## ⏯️ Возобновить воспроизведение

MoviePlayer автоматически запоминает позицию, на которой вы остановились. При следующем воспроизведении того же видео Enigma2 спросит вас, хотите ли вы продолжить с того места, где остановились.


**Поддерживается для:**
- ✅ Локальных файлов (HDD, USB)
- ✅ Сетевых ресурсов (SMB/NFS)
- ✅ HTTP-потоков (Open Directory)
- ✅ Плейлистов M3U и .tv

## 🚀 Как это работает

**Старый способ (версия 1.2 и ранее):**

ref = eServiceReference(4097, 0, path)
session.nav.playService(ref) # Нет управления, нет возобновления воспроизведения

Новый способ (версия 1.3):

ref = eServiceReference(4097, 0, path)
ref.setName(name)
session.open(MoviePlayer, ref) # Полный контроль, поддержка возобновления воспроизведения!
