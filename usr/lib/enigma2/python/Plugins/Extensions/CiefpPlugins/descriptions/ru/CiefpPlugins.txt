..:: Панель плагинов Ciefp ::..

Ciefp Plugins — это плагин для Enigma2, предоставляющий пользовательский интерфейс для просмотра, установки и управления коллекцией плагинов Ciefp.
Он позволяет пользователям легко устанавливать такие плагины, как Ciefpsettings Motor, Ciefp IPTV Bouquets, Ciefp EPGshare и другие,
просматривать их описания на нескольких языках, просматривать изображения и проверять наличие обновлений.
Плагин разработан для удобной навигации с упором на функциональность и лаконичный внешний вид.

Как использовать
Запуск плагина:
Найдите Ciefp Plugins в меню Enigma2 в разделе «Плагины» и запустите его.
Откроется экран CiefpPluginsPanel со списком плагинов слева, описанием справа и цветными кнопками внизу.

Навигация:
Левая сторона (список плагинов): используйте стрелки вверх/вниз для навигации по списку плагинов. У каждого плагина есть значок и название.
Правая сторона (описание): Отображает подробное описание выбранного плагина на текущем языке (например, название: Ciefpsettings Engine и текст описания).

Фокус: Нажмите OK, чтобы переключить фокус между списком плагинов и описанием (чтобы прокрутить описание с помощью стрелок).

Функции кнопок:
Красная (Выход): Выход из плагина.
Зелёная (Установить): Установка выбранного плагина. Подтвердите установку в диалоговом окне.
Жёлтая (Язык): Изменение языка описания плагина (например, en, sr, el, de, sk).
Синяя (Просмотр изображений): Открывает экран просмотра изображений для просмотра изображений выбранного плагина (если изображения есть в /usr/lib/enigma2/python/Plugins/Extensions/CiefpPlugins/pictures/).

Просмотр изображений:
Синяя кнопка открывает экран просмотра изображений плагина.
Используйте зелёную (Предыдущая), синюю (Далее) или стрелки влево/вправо для навигации по изображениям.
Красная кнопка или кнопка «Отмена» возвращают на главный экран.

Сообщения о состоянии:
Сообщения о состоянии (например, «Проверка обновлений», «Выполняется установка», «Установка выполнена успешно»)
отображаются в правой нижней части экрана, рядом с цветными кнопками, в виджете status_label.

Автоматическая проверка обновлений:
При запуске плагин автоматически проверяет наличие новой версии по адресу VERSION_URL.

Если она есть (например, версия 2.0), он спрашивает, хотите ли вы обновиться. Обновление включает в себя резервное копирование списка установленных плагинов и перезапуск Enigma2.

Что нового в версии 1.9
Поддержка языков (en, sr, el, de, sk):
Плагин динамически загружает языки из папки /usr/lib/enigma2/python/Plugins/Extensions/CiefpPlugins/descriptions/.
Поддерживаются следующие языки: en (английский), sr (сербский), el (греческий), de (немецкий) и sk (словацкий). Также можно добавлять другие языки, создавая новые папки.

Описания плагинов загружаются на выбранном языке, а если нужный язык недоступен, то по умолчанию используется en.
Смена языка осуществляется с помощью жёлтой кнопки, название языка пишется с заглавной буквы (например, Sr для сербского).

Автоматическая проверка обновлений:
Плагин проверяет наличие новой версии при запуске, используя VERSION_URL.
Если доступна новая версия, она предлагает возможность обновления с помощью резервной копии списка установленных плагинов (INSTALLED_PLUGINS_FILE) в /tmp/ciefp_plugins_backup.txt.
Сообщения о состоянии в status_label:
Новое добавление: Сообщения о состоянии (например, «Проверка обновлений», «Установка выполнена успешно», «Обновление не удалось») отображаются в отдельном виджете справа от синей кнопки, а не перекрывают описание плагина.
Это улучшает пользовательский интерфейс, поскольку описание плагина остаётся видимым во время работы.

Список плагинов: Отображает 23 плагина (например, Ciefpsettings Motor, Ciefp IPTV Bouquets, Ciefp EPGshare) с иконками и описаниями.
Многоязыковая поддержка: Автоматически загружает языки из папки descriptions/, с текущей поддержкой en, sr, el, de, sk.
Просмотр изображений: Позволяет просматривать изображения плагинов на отдельном экране.
Установка плагина: простая установка с помощью скриптов с GitHub.

Автоматические обновления: проверка наличия и установка новых версий плагина.
