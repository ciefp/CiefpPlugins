..:: CiefpE2Converter ::..

CiefpE2Converter — это плагин для Enigma2, предназначенный для преобразования плейлистов M3U в пакеты IPTV, которые можно использовать на устройствах Enigma2.
Позволяет пользователям легко выбирать группы или отдельные каналы из M3U-файлов, задавать имя пакета и выбирать нужный тип сервиса
(например, Gstreamer, Exteplayer3, Streamlink).
Плагин генерирует соответствующие файлы «userbouquet» и регистрирует их в bouquets.tv для немедленного использования.

Новости версии 1.9:
Поддержка M3U-ссылок в формате playlist.txt:
Плагин теперь поддерживает загрузку M3U-ссылок из текстовых файлов (playlist.txt).
Пользователи могут выбрать ссылку из списка, который автоматически загружается и обрабатывается для дальнейшей обработки.

Новый экран выбора (ChannelSelectionScreen):
Добавлен новый экран выбора каналов, если M3U-файл не содержит групп.
Этот экран позволяет пользователям выбирать отдельные каналы или группы сериалов,
с автоматической группировкой эпизодов под одним названием (например, «Секретные материалы» вместо «Секретные материалы S01 E01»).

Новая кнопка «Выбрать похожие»:
Реализована новая кнопка «Выбрать похожие» на экранах выбора групп (GroupSelectionScreen) и выбора каналов (ChannelSelectionScreen).
Функция: Автоматически выбирает все элементы (группы или каналы), имеющие общий префикс с текущим выбранным элементом. Например:
Для группы «Sport TV 1» выбираются все группы, начинающиеся с «Sport TV» (например, «Sport TV 2», «Sport TV HD»).
Для канала «HBO HD» выбираются все каналы, начинающиеся с «HBO» (например, «HBO 2», «HBO 3»).
Это позволяет быстро выбирать похожие элементы без необходимости вручную отмечать каждый элемент отдельно.

Другие характеристики:
Интуитивно понятный интерфейс с возможностью просмотра файлов в каталоге /tmp.
Настройка имени пакета с помощью виртуальной клавиатуры.
Поддержка различных типов сервисов (Gstreamer, Exteplayer3, DVB, Radio, Streamlink).
Автоматическая регистрация пакетов и возможность перезагрузки настроек после конвертации.
Этот плагин идеально подходит для пользователей, которые хотят быстро и эффективно организовать свой IPTV-контент на платформе Enigma2,
добавив дополнительную гибкость благодаря новым функциям версии 1.7.

Краткая инструкция по использованию плагина CiefpE2Converter v1.7

Запуск плагина:
Найдите CiefpE2Converter v1.7 в меню плагинов на вашем устройстве Enigma2 и запустите его.

Выбор M3U-файла или ссылки:
На главном экране (MainScreen) вы увидите список файлов в каталоге /tmp.
Если у вас есть M3U-файл: перейдите к нему (например, playlist.m3u) и нажмите OK. Если у вас есть файл playlist.txt со ссылками: выберите файл .txt и нажмите «ОК». Откроется экран выбора плейлиста:
Прокрутите страницу вверх/вниз и выберите нужную ссылку (нажмите «ОК» или зелёную кнопку «Подтвердить»).
Плагин загрузит M3U-файл по этой ссылке и продолжит работу.

Название группы:
Нажмите жёлтую кнопку «Определить имя», чтобы открыть виртуальную клавиатуру.
Введите название группы (например, «Мой IPTV») и подтвердите, нажав «ОК». Имя по умолчанию — «IPTV Mix Bouquet».

Выбор групп или каналов:
Нажмите синюю кнопку «Выбрать группы»:
Если в M3U-файле есть группы: откроется экран выбора групп. Отметьте нужные группы:
OK: Включает/удаляет группу.
Выбрать все (синяя кнопка): Указывает на все группы. Выбрать похожие (жёлтая кнопка): обозначает группы с похожим префиксом (например, «Спортивное ТВ 1» -> «Спортивное ТВ 2»).

Если в M3U нет групп, откроется экран выбора каналов. Теги каналов или сериалов:
OK: включает/выключает канал/сериал.
Выбрать все (синяя кнопка): обозначает все каналы.
Выбрать похожие (жёлтая кнопка): обозначает каналы с похожим префиксом (например, «HBO HD» -> «HBO 2»).

Нажмите зелёную кнопку «Подтвердить» для подтверждения или красную кнопку «Отмена» для отмены.

Преобразование в букет:
На главном экране нажмите зелёную кнопку «Преобразовать».
Выберите тип сервиса (например, «Gstreamer (4097:0:1)») из списка и подтвердите, нажав OK.
Плагин создаёт букет в /etc/enigma2 и регистрирует его в bouquets.tv.

Перезагрузить настройки:
После конвертации появится вопрос «Перезагрузить настройки?». Выберите «Да», чтобы обновить список композиций, или «Нет», чтобы выйти.

Выход:
Нажмите красную кнопку «Выход», чтобы закрыть плагин.

Примечание:
Загруженные файлы M3U сохраняются в /tmp/m3u_playlist.
Сериалы на экране выбора каналов ChannelSelectionScreen сгруппированы (например, «Секретные материалы» включают все серии, если они отмечены).
Проверьте, доступен ли каталог /tmp и достаточно ли места для загрузки файлов.
Это всё, что вам нужно знать для базового использования плагина!
