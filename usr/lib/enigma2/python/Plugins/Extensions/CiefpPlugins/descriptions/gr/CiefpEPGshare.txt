..:: CiefpEPGshare ::..

Πρόσθετο Enigma2 για λήψη και εμφάνιση δεδομένων EPG (Ηλεκτρονικός Οδηγός Προγράμματος) από τον ιστότοπο epgshare01.online
Επιτρέπει στους χρήστες να επιλέγουν αρχεία XML με δεδομένα EPG, να εμφανίζουν κανάλια και τα προγράμματά τους, καθώς και εικονίδια καναλιών.
Το πρόσθετο είναι βελτιστοποιημένο για απόδοση, με προσωρινή αποθήκευση δεδομένων και φιλτράρισμα μεγάλων αρχείων,
και περιλαμβάνει μια προσαρμοσμένη διεπαφή χρήστη με λογότυπα για καλύτερη οπτική εμπειρία.

Επιλογή αρχείου XML:
Στην οθόνη XMLSelectionScreen (ανάλυση 1280x852) βλέπετε μια λίστα με τα διαθέσιμα αρχεία .xml.gz.
Χρησιμοποιήστε τα βέλη πάνω και κάτω στο τηλεχειριστήριο για να πλοηγηθείτε στη λίστα.
Πατήστε OK για να επιλέξετε ένα αρχείο και να μεταβείτε στην οθόνη δεδομένων EPG (CiefpEPGshare).

Προβολή δεδομένων EPG:
Στην οθόνη CiefpEPGshare (ανάλυση 1800x800) βλέπετε:
Λίστα καναλιών (αριστερά): Λίστα καναλιών από το επιλεγμένο αρχείο XML.
Πληροφορίες EPG (μεσαία): Προγράμματα για το επιλεγμένο κανάλι, ταξινομημένα κατά ημερομηνία και ώρα.
Εικονίδιο και λογότυπα: Εμφανίζει το εικονίδιο καναλιού (εικονίδιο), το λογότυπο του πρόσθετου και τις εικόνες φόντου.

Χρησιμοποιήστε τα βέλη πάνω και κάτω για να πλοηγηθείτε στα κανάλια ή στα δεδομένα EPG.
Πατήστε OK για να αλλάξετε την εστίαση μεταξύ της λίστας καναλιών και των πληροφοριών EPG.
Πατήστε Άκυρο για να επιστρέψετε στην προηγούμενη οθόνη ή να κλείσετε.

Αγνόηση αρχείων XML:
Το αρχείο epg_ripper_ALL_SOURCES1.xml.gz αγνοείται αυτόματα λόγω του μεγέθους του (1,3 GB, 22 εκατομμύρια γραμμές) για την αποφυγή προβλημάτων απόδοσης.

Για να προσθέσετε άλλα αρχεία XML στη λίστα αγνόησης:
Στην οθόνη XMLSelectionScreen, επιλέξτε το αρχείο που θέλετε να αγνοήσετε.
Πατήστε το κόκκινο κουμπί στο τηλεχειριστήριο.
Το αρχείο θα προστεθεί στο /tmp/CiefpEPGshare/ignore_list.txt και θα εξαιρεθεί από την οθόνη.
Τα αγνοημένα αρχεία δεν εμφανίζονται στη λίστα μέχρι να τα αφαιρέσετε χειροκίνητα από το ignore_list.txt (π.χ. μέσω SSH ή FTP).

Αρχεία καταγραφής και διαγνωστικά:
Εάν προκύψουν σφάλματα (π.χ. προβλήματα φόρτωσης δικτύου ή λογότυπου), ελέγξτε το /tmp/ciefp_epgshare.log για λεπτομέρειες.
Το πρόσθετο προσπαθεί αυτόματα να εγκαταστήσει τη βιβλιοθήκη lxml εάν λείπει ή χρησιμοποιεί το ElementTree ως αντικατάσταση.
