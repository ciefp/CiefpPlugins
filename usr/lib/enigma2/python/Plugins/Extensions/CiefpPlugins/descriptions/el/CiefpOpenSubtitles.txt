🎬 Πρόσθετο CiefpOpenSubtitles v1.1
Πρόσθετο αναζήτησης και λήψης υποτίτλων για αποκωδικοποιητές που βασίζονται στο Enigma2 (OpenATV, PurE2, OpenPLi, κ.λπ.)
— υποστήριξη OpenSubtitles.com και OpenSubtitles.org.

✅ Βασικά Χαρακτηριστικά:
🔍 Αναζήτηση υπότιτλων για ταινίες και τηλεοπτικές σειρές ανά τίτλο
🌐 Υποστήριξη διπλής τοποθεσίας:
OpenSubtitles.com (μέσω δωρεάν κλειδιού API — συνιστάται)
OpenSubtitles.org (μέσω σύνδεσης με όνομα χρήστη/κωδικό πρόσβασης)
🗣️ Υποστήριξη πολλαπλών γλωσσών: εισαγάγετε κωδικούς γλώσσας (π.χ. sr, hr, en ή srp, hrv, eng)
⬇️ Αυτόματη λήψη και αποθήκευση σε κατάλογο που ορίζεται από τον χρήστη
🔄 Λειτουργία λήψης πολλαπλών γλωσσών — ανάκτηση υπότιτλων σε όλες τις διαμορφωμένες γλώσσες για έναν δεδομένο τίτλο

🛠️ Πλήρως διαμορφώσιμο μέσω της οθόνης ρυθμίσεων:
Διαδρομή αποθήκευσης
Καθυστέρηση λήψης (για τήρηση των ορίων ρυθμού API)
Προτεραιότητα γλώσσας (πρώτη αντιστοίχιση έναντι όλων των γλωσσών)
Αυτόματη μετονομασία για αντιστοίχιση αρχείων βίντεο (π.χ. MovieName.sr.srt)

📱 Τρόπος χρήσης:
Εκκινήστε το πρόσθετο από το μενού Επεκτάσεις.
Επιλέξτε "Αναζήτηση Ταινιών" ή "Αναζήτηση Σειρών".
Πατήστε ΚΙΤΡΙΝΟ για να ανοίξετε το εικονικό πληκτρολόγιο και πληκτρολογήστε τον τίτλο.
Πατήστε ΠΡΑΣΙΝΟ για να ξεκινήσετε την αναζήτηση.
Πλοηγηθείτε στα αποτελέσματα με τα πλήκτρα βέλους.
Πατήστε ΜΠΛΕ ή OK για να κατεβάσετε τους επιλεγμένους υπότιτλους.
Οι υπότιτλοι αποθηκεύονται αυτόματα (προεπιλογή: /media/hdd/subtitles/).

💡 Σημείωση: Για το OpenSubtitles.com, χρειάζεστε ένα δωρεάν κλειδί API (εγγραφείτε στη διεύθυνση https://www.opensubtitles.com).

🛠️ Αποθήκευση Κλειδιού API, Όνομα Χρήστη και Κωδικού Πρόσβασης
Στο κύριο μενού του πρόσθετου, επιλέξτε Κλειδί API / Σύνδεση (μπλε/OK).
Για .com (συνιστάται - ταχύτερο και καλύτερο):

Πατήστε ΠΡΑΣΙΝΟ (Επεξεργασία Κλειδιού API).
Εισαγάγετε το κλειδί API από το opensubtitles.com (δωρεάν λογαριασμός > καρτέλα API > Δημιουργία νέου κλειδιού).
Αποθηκεύεται αυτόματα στο αρχείο: /etc/enigma2/ciefpopensubtitles/opensubtitles_apikey.txt
(μορφή: apikey=xxxxxxxxxxxxxxxxx).

Για .org (παλαιότερο σύστημα):
Πατήστε ΚΙΤΡΙΝΟ (Επεξεργασία σύνδεσης).
Εισαγάγετε το όνομα χρήστη και, στη συνέχεια, τον κωδικό πρόσβασης από το opensubtitles.org.
Αποθηκεύεται αυτόματα στο αρχείο: /etc/enigma2/ciefpopensubtitles/opensubtitles_login.txt
(μορφή:
user=yourusername
pass=yourpassword).

🇬🇧 Πώς να αποκτήσετε ένα κλειδί API (OpenSubtitles.com)
Μεταβείτε στη διεύθυνση https://www.opensubtitles.com
Κάντε κλικ στην επιλογή "Σύνδεση" (ή "Εγγραφή" αν δεν έχετε λογαριασμό)
Μετά τη σύνδεση, μεταβείτε στη διεύθυνση: Λογαριασμός → API
Κάντε κλικ στην επιλογή "Δημιουργία διακριτικού API"
Αντιγράψτε το δημιουργημένο διακριτικό API (π.χ. abc123xyz456)
Εισαγάγετε το στο πρόσθετο:
Επεκτάσεις → OpenSubtitles → Ρυθμίσεις → API/Σύνδεση → διακριτικό API
🔐 Το διακριτικό API είναι προσωπικό — μην το κοινοποιείτε.
