..:: CiefpSettingsMotor ::..

Что делает плагин?
CiefpSettingsMotor — это простой инструмент, который загружает, распаковывает и устанавливает актуальный список каналов для моторизованных спутниковых антенн
прямо с GitHub (ciefp-E2-75E-34W). Этот список включает спутниковые пакеты (например, 19.2E, 16.0E, 13.0E...),
тематические пакеты (спорт, фильмы, музыка), каналы IPTV и маркеры, адаптированные для широкого диапазона положений спутников.

Как использовать?
1. Запустите плагин из меню ресивера Enigma2.
2. Нажмите «OK» на пульте дистанционного управления.
3. Дождитесь загрузки и установки списка — процесс автоматический и занимает всего несколько минут.
4. После установки ресивер автоматически обновит страницу, и новый список станет активным.

Что нового в версии 2.2?
• Сохраняет пользовательские букеты: ваши персонально созданные букеты (например, userbouquet.moje_favorite.tv) не будут удалены.
Плагин сохраняет их и помещает в конец списка в bouquets.tv, а новые спутниковые, тематические и IPTV букеты добавляются в начало.
• Полная установка: все файлы из списка (lamedb, .tv, .radio) копируются в /etc/enigma2/,
а satellites.xml — в /etc/tuxbox/, обеспечивая полную актуальность.
• Простота: никаких сложных настроек — всё работает одним кликом.

Для кого предназначен?
Этот плагин идеально подходит для пользователей с моторизованными антеннами, которым нужно обновлять список каналов без потери пользовательских настроек.
Установка быстрая и простая как для новичков, так и для опытных пользователей.
