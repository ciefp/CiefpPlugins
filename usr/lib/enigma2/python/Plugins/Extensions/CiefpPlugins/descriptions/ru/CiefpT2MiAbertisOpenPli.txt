..:: CiefpSettings T2MI Abertis OpenPLi ::...

Это адаптированная версия для OpenPLi,
которая позволяет устанавливать и настраивать дополнения, необходимые для использования T2MI и настроек спутников Abertis.

Изменения в новой версии:

Увеличенный экран:
Размер экрана изменен с 1200x600 до 1600x800,
изменены положение и размеры виджетов (информация, статус, кнопки), а также увеличен размер шрифта для лучшей читаемости.

Локальные файлы:
Мы отменили загрузку файлов из интернета (кроме astra-sm)
и перешли на копирование файлов из локальных каталогов (data и data/scripts) с помощью Shutil.copy.

Плагин автоматически устанавливает:
Astra-SM: инструмент для обработки сигналов T2MI. (этот файл скачан из интернета)
Файлы конфигурации: astra-sm.conf, astra-sm.lua и sysctl.conf, SoftCam.Key, скрипт abertis (теперь скопирован из локальной директории)
Интеграция с oscam-emu.
Этот плагин упрощает установку необходимых компонентов и предназначен
для пользователей, которые хотят быстро и легко настроить OpenPLi для T2MI/Abertis.

Настройки DNS-сервера:
Для корректной работы сети при использовании статического IP-адреса необходимо вручную настроить DNS-серверы.

Эти DNS-серверы принадлежат Google и обеспечивают надежное разрешение доменных имен.
Это делается через настройки сети:
nameserver 8.8.8.8
nameserver 8.8.4.4
Перезапуск сети: После внесения изменений обязательно перезапустите сеть:

При правильной настройке сеть со статическим IP-адресом работает без проблем, и плагин может загружать необходимые файлы из интернета.
