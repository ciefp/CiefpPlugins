- Enigma2 Music Jukebox Plugin

### Description:
CiefpVibes is a great music player plugin for Enigma2 boxes.
Plays local or online playlists (.m3u/.tv) with repeat, shuffle,
custom backgrounds and integration with GitHub for fresh songs.
Supported mp3 and flac formats.
Ideal for non-stop vibes!

### Usage:
Start from the Plugins menu.
Green: Load local file.
Blue: Online from GitHub.
Yellow: Settings.
Red: Exit.
Navigate the playlist, OK to play.
Automatically saves the last session.

Feel the good vibes with the CiefpVibes enigma2 plugin
