..:: Плагин CiefpSettings T2mi Abertis ::..

Этот плагин предназначен для простой установки необходимых конфигураций и скриптов
для использования функций T2MI Abertis на устройствах Enigma2.

Что делает плагин?

- Автоматически проверяет совместимость системы (Python 3 и процессор ARM).
- Устанавливает **Astra-SM** через `opkg`.
- Загружает и размещает файлы конфигурации:
- **astra.conf** в `/etc/astra/`.
- Скрипт **abertis** в `/etc/astra/scripts/`.
- **softcam.key** в двух папках: `/etc/tuxbox/config/` и `/etc/tuxbox/config/oscam-emu/`.
- **sysctl.conf** в `/etc/`.
- В конце установки отображается обзор установленных файлов и предлагается перезагрузить систему.

Установка выполнена успешно! Установленные файлы:
- astra-sm
- astra.conf
- abertis
- softcam.key (/etc/tuxbox/config/)
- softcam.key (/etc/tuxbox/config/oscam-emu/)

Установка завершена. Перезагрузите систему.

Зачем использовать этот плагин?
Автоматическая установка и настройка без необходимости ручного вмешательства.
Быстрая и простая поддержка сервисов Abertis и T2MI.

Примечание:
Эта версия поддерживает только процессоры ARM и предназначена для устройств на базе Python 3.
Установка выполняется через Telnet с помощью команды wget.
