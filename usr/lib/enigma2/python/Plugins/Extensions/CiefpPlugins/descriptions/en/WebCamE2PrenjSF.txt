..:: WebCamE2PrenjSF ::..

py2 & py3 support
WebCamE2PrenjSF is a simple Enigma2 plugin that allows quick download
and installation of the latest userbouquet with webcams directly from the GitHub repository.

- Automatically checks the bouquet version and downloads the latest one if there is a newer one than the installed one.

- The installed bouquet is stored in /etc/enigma2/ and registered in bouquets.tv.

- After installation, it offers an option to automatically reload the channel list.
- It is very easy to use â€“ run the plugin and press OK.

The plugin is intended for everyone who wants to always have an updated list of webcams
on their Enigma2 receiver without manually copying files.
The plugin was made with the joint efforts of Prenj and Ciefp Satellite Forum ( SF )
