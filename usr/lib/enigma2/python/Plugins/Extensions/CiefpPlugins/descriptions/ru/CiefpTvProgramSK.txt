..:: CiefpTvProgramSK ::..

CiefpTvProgramSK — плагин для Enigma2, предназначенный для отображения электронного программного гида (EPG) для телеканалов,
используя данные, загруженные с https://epgshare01.online/epgshare01/epg_ripper_SK1.xml.gz.
Плагин позволяет пользователям просматривать список каналов и подробную информацию о EPG, включая названия передач, время трансляции, описания и категории.
Плагин поддерживает отображение пиктограмм (значков каналов), логотипов плагинов и фоновой графики, а также кэширование данных для более быстрой загрузки.
Интерфейс интуитивно понятен, с навигацией между списком каналов и информацией о EPG, и адаптирован для платформы Enigma2.

Использование плагина

Навигация:
Список каналов: В левой части экрана отображается список доступных каналов, загруженных из XML-файла.
Информация о EPG: В правой части экрана отображается информация о программе выбранного канала (время, название, описание, категория).
Используйте клавиши «Вверх»/«Вниз» для навигации по списку каналов или информации о EPG в зависимости от текущего фокуса.
Нажмите «OK», чтобы переключить фокус между списком каналов и информацией о EPG.
Нажмите «Отмена» (или «Выход»), чтобы выйти из плагина.
Функции:
Отображение значка: Значок выбранного канала отображается внизу экрана, если он доступен в папке
/usr/lib/enigma2/python/Plugins/Extensions/CiefpTvProgramSK/picon.
Если значок не найден, используется значок по умолчанию (placeholder.png).

Кэширование данных: Данные EPG кэшируются в файле /tmp/CiefpProgramSK/epg_cache.xml на 24 часа для уменьшения объёма загрузки данных.
Ведение журнала: Плагин регистрирует действия и ошибки в файле /tmp/ciefp_tvprogramsk.log для диагностики.
