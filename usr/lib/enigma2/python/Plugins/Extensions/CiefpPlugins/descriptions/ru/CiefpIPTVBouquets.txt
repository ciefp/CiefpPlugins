..:: CiefpIPTVBouquets ::..

CiefpIPTVBouquets — плагин для Enigma2, предназначенный для управления наборами каналов IPTV на устройствах Enigma2.
Он позволяет пользователям загружать, устанавливать, редактировать и удалять наборы каналов IPTV непосредственно из репозитория GitHub,
а также управлять каналами внутри набора. Плагин полезен для пользователей, желающих легко организовать контент IPTV,
включая каналы, сериалы и фильмы, с помощью интуитивно понятного интерфейса.

Основные функции:

Загрузка и установка наборов каналов IPTV с GitHub.

Редактирование набора каналов (выбор, перемещение, удаление каналов).

Групповой выбор похожих каналов (например, «HBO 1», «HBO 2», «HBO»).

Управление удалёнными наборами каналов (файлы .del).

Сброс настроек после внесения изменений.

Экраны и их использование
Плагин имеет один основной экран и четыре дополнительных экрана, каждый из которых имеет определённые функции.
Вот краткое описание и инструкции по использованию каждого из них:

1. ..:: CiefpIPTV ::.. (Главный экран)
Описание: Этот экран позволяет загружать и устанавливать пакеты IPTV из репозитория GitHub.

Функции:
- Отображает список доступных пакетов из GitHub (слева).
- Позволяет выбрать пакет для установки (справа).
- Начинает установку выбранных пакетов и обновляет настройки.
- Предоставляет доступ к другим экранам (IPTV Manager, BouquetViewer).

Использование:
- OK/Зелёный: Выбирает/отменяет выбор пакета из левого списка и добавляет его в правый список.
- Жёлтый: Начинает установку выбранных пакетов (подтверждается диалоговым окном).
- Красный: Открывает экран IPTV Manager.
- Синий: Открывает экран BouquetViewer.
- Отмена: Выход из плагина.
- Вверх/Вниз: Навигация по списку пакетов.
Примечание: Букеты загружаются с GitHub, поэтому требуется подключение к Интернету.

2. ..:: BouquetViewer ::..
Описание: Позволяет просмотреть выбранный букет перед установкой.

Функции:
- Отображение содержимого выбранного букета даёт пользователю возможность просмотреть выбранный букет перед установкой.

3. ..:: IPTV Manager ::..
Описание: Позволяет управлять установленными букетами IPTV, включая удаление и редактирование.
Функции:
- Отображает список установленных букетов IPTV.
- Позволяет выбирать и удалять букеты.
- Предоставляет доступ к редактированию букетов через экран редактора IPTV.

Применение:
- OK/Зелёный: Выбирает/отменяет выбор букета для удаления.
- Красный: Удаляет выбранные букеты (подтверждается в диалоговом окне).
– Жёлтый: открывает инструмент «Очистка букета» для удаления del-файлов из etc/enigma2.
– Синий: открывает редактор IPTV для редактирования выбранного букета.
– Отмена: возвращает на главный экран.
– Вверх/Вниз: перемещает по списку букетов.

Примечание: удаление букетов удаляет их из системы, и перезагрузка настроек после удаления необязательна.

3. ..:: Редактор IPTV ::..
Описание: Позволяет детально редактировать каналы в выбранном пакете.
Функции:
- Отображение списка каналов в пакете.
- Поддержка выбора, группового выбора похожих каналов, перемещения и удаления каналов.
- Позволяет сохранять изменения и перезагружать настройки.
Применение:
- OK: Выбор/отмена выбора отдельного канала.
- Красный: Удаление выбранных каналов (подтверждается диалоговым окном).
- Зелёный: Сохранение изменений в пакете (подтверждается диалоговым окном перезагрузки).
- Жёлтый: Включение/выключение режима перемещения для перемещения выбранных каналов.
- Синий: Запуск группового выбора похожих каналов (например, выбор «HBO 1» означает «HBO», «HBO 2» и т. д.).
- Вверх/Вниз: Навигация по списку каналов; в режиме перемещения перемещает выбранные каналы.
- Влево/Вправо: Прокрутка страниц; В режиме перемещения перемещает каналы на несколько позиций.
- Отмена: выход из окна (предупреждение о наличии несохранённых изменений).
Примечание: функция «Выбрать похожие» оптимизирована для распознавания похожих каналов (например, «HBO», «Cinemax», «Звёздные войны: Эпизод»),
включая отмену выбора, если все похожие каналы уже выбраны.

4. ..:: Bouquet Cleaner ::..
Описание: управление удалёнными букетами (файлами с расширением .del).
Функции:
- Отображение списка файлов .del в каталоге /etc/enigma2/.
- Возможность выбора и безвозвратного удаления файлов .del.
Применение:
- OK: выбор одного файла .del.
- Зелёный: выбор всех файлов .del.
- Красный: удаление выбранных файлов .del (подтверждается диалоговым окном).
- Отмена: возврат на главный экран.
- Вверх/Вниз: перемещение по списку файлов.

Примечание: полезно для очистки старых или ненужных удалённых наборов.

Как использовать плагин
Установка набора:
Запустите плагин и дождитесь загрузки наборов с GitHub на главном экране.
Выберите нужные наборы (OK/Зелёный) и установите их (Жёлтый).
При необходимости подтвердите перезагрузку настроек.

Просмотр наборов:
Просмотр содержимого выбранного набора перед установкой

Редактирование набора:
На главном экране перейдите в менеджер IPTV (Красный).
Выберите набор и откройте редактор IPTV (Синий).
В редакторе IPTV выберите каналы (OK), сгруппируйте похожие каналы (Синий), переместите их (Жёлтый + Вверх/Вниз) или удалите (Красный).
Сохраните изменения (Зелёный) и подтвердите перезагрузку.

Удаление группы каналов:
В диспетчере IPTV выберите группы каналов (кнопка «ОК»/зелёная) и удалите их (красная).
Подтвердите перезагрузку настроек.

Очистка del-файлов:
На главном экране перейдите в раздел «Очистка» (синяя кнопка).
Выберите del-файлы (кнопка «ОК» или зелёная кнопка для всех каналов) и удалите их (красная кнопка).

Заключение
CiefpIPTVBouquets — это мощный инструмент для управления группами каналов IPTV, отличающийся простотой и гибкостью.
Главный экран предназначен для установки, диспетчер IPTV — для управления группами каналов, редактор IPTV — для детального редактирования каналов, а инструмент «Очистка» — для удаления удалённых файлов.
