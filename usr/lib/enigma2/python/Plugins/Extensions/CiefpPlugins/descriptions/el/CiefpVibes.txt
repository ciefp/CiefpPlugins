- Πρόσθετο Enigma2 Music Jukebox

### Περιγραφή:
Το CiefpVibes είναι ένα εξαιρετικό πρόσθετο αναπαραγωγής μουσικής για τα Enigma2.

Αναπαράγει τοπικές ή online λίστες αναπαραγωγής (.m3u/.tv) με επανάληψη, τυχαία αναπαραγωγή,
προσαρμοσμένα φόντα και ενσωμάτωση με το GitHub για νέα τραγούδια.

Υποστηριζόμενες μορφές mp3 και flac.
Ιδανικό για ασταμάτητη μουσική!

### Χρήση:
Έναρξη από το μενού Πρόσθετων.

Πράσινο: Φόρτωση τοπικού αρχείου.

Μπλε: Online από το GitHub.

Κίτρινο: Ρυθμίσεις.

Κόκκινο: Έξοδος.
Πλοηγηθείτε στη λίστα αναπαραγωγής, OK για αναπαραγωγή.

Αποθηκεύει αυτόματα την τελευταία συνεδρία.

Νιώστε την καλή ατμόσφαιρα με το πρόσθετο CiefpVibes enigma2

### Νέα επιλογή - ΡΑΔΙΟΦΩΝΟ
Πατήστε 🔵 ΜΠΛΕ - Online λίστες
Επιλέξτε ΡΑΔΙΟΦΩΝΟ, τη δυνατότητα επιλογής διαφόρων λιστών αναπαραγωγής

Προστέθηκε GITHUB_RADIO_URL και υποστήριξη για αρχεία .radio (σε ανάλυση, πρόγραμμα περιήγησης αρχείων και λήψη GitHub).
Βελτιωμένη ανάλυση αρχείων .tv/.radio: Εκτεταμένη ανίχνευση ροών ήχου
(ενδείξεις όπως .mp3, .flac, /stream, rtmp://), με εξαιρέσεις για μορφές βίντεο (.ts, .mp4, κ.λπ.).
Καλύτερη ανάλυση μεταδεδομένων: parseArtistTitle() για διαχωρισμό της μορφής "καλλιτέχνης - τίτλος",
υποστήριξη για μεταδεδομένα ICY, αυτόματη ενημέρωση κειμένου και αφισών που αναπαράγονται τώρα.

### Νέα επιλογή - Εκκαθάριση προσωρινής μνήμης:
Πατήστε 🟡 ΚΙΤΡΙΝΟ (Ρυθμίσεις)

Επιβεβαιώστε τη διαγραφή στο παράθυρο διαλόγου
Η προσωρινή μνήμη θα διαγραφεί (αφίσες + λίστες)

### Βελτιώσεις οπτικής και αναπαραγωγής:
- Προστέθηκε δυναμική εμφάνιση αφισών (showPoster(), showDefaultPoster())
με αυτόματη λήψη του εξωφύλλου του άλμπουμ από το iTunes API (fetchAlbumCover()).
- Δύο γραμμές προόδου: "progress_real" (ακριβής πρόοδος) και "progress_vibe" (παλμική πράσινη γραμμή για εφέ "vibe").
- Εντοπισμός εκτός σύνδεσης: Ετικέτα για "offline_status" (π.χ. "Stream Offline"),
χρονοδιακόπτης για έλεγχο της ροής (checkStreamStatus()), επαναφορά κατάστασης κατά την αναπαραγωγή.
- Περισσότερες επιλογές εμφάνισης: Έως 10 φόντα, αφίσες και γραμμές πληροφοριών (μόνο στην έκδοση 1.0 έως 5).
- Αυτόματη αναπαραγωγή του πρώτου τραγουδιού στη λίστα αναπαραγωγής καλωσορίσματος.

1. ΤΙ ΠΡΕΠΕΙ ΝΑ ΓΙΝΕΙ ΣΤΟΝ ΦΟΡΗΤΟ ΥΠΟΛΟΓΙΣΤΗ/ΦΟΡΗΤΟ ΤΟΠΟ:

Για Windows:
bash
1. Κάντε δεξί κλικ στον φάκελο → "Ιδιότητες"
2. Καρτέλα "Κοινή χρήση" → "Κοινή χρήση για προχωρημένους"
3. ✅ Κοινή χρήση tαυτόν τον φάκελο
4. "Δικαιώματα" → Προσθήκη "Όλων" → Πλήρης έλεγχος
5. Κατάργηση όλων των αιτημάτων κωδικού πρόσβασης
6. Κέντρο δικτύου και κοινής χρήσης → Απενεργοποίηση της κοινής χρήσης με προστασία κωδικού πρόσβασης

Για Linux/Mac:
bash
# Εγκατάσταση Samba εάν δεν είναι εγκατεστημένο
sudo apt install samba
sudo nano /etc/samba/smb.conf

# Προσθήκη στο τέλος:
[Μουσική]
διαδρομή = /διαδρομή/προς/φάκελο
browseable = ναι
μόνο ανάγνωση = ναι
guest ok = ναι
αναγκαστική χρήση χρήστη = κανείς

# Επανεκκίνηση Samba
sudo systemctl restart smbd

2. ΠΡΟΒΛΗΜΑΤΑ ΣΥΝΔΕΣΗΣ ΚΑΙ ΛΥΣΕΙΣ:
Λύση προβλήματος
"Δεν επιτρέπεται η πρόσβαση" Κατάργηση της κοινής χρήσης με προστασία κωδικού πρόσβασης
"Δεν βρέθηκε διαδρομή δικτύου" Έλεγχος τείχους προστασίας και διεύθυνσης IP
"Σφάλμα προσάρτησης" Χρησιμοποιεί την επιλογή guest αντί για user/pass
Αργή σύνδεση Μείωση χρονικού ορίου στις ρυθμίσεις του πρόσθετου

3. ΤΑ ΠΙΟ ΣΥΧΝΑ ΧΡΗΣΙΜΟΠΟΙΟΥΜΕΝΑ ΔΙΑΔΡΟΜΗ:
bash
//192.168.1.100/Μουσική # Windows χωρίς έλεγχο ταυτότητας
//192.168.1.100/ # Κοινόχρηστο στοιχείο root
//namelaptopa/Public # Δημόσιος φάκελος

Σύστημα αυτόματης ενημέρωσης: Προστέθηκε έλεγχος έκδοσης μέσω του αρχείου version.txt στο GitHub.

Το πρόσθετο πλέον εντοπίζει αυτόματα νέες εκδόσεις, εμφανίζει την κατάσταση στην οθόνη (π.χ. "Ενημερωμένη έκδοση ✓" ή "Διαθέσιμη ενημέρωση v1.5!")
