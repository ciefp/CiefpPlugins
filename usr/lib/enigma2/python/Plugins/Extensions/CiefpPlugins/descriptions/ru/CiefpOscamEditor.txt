..:: Ciefp Oscam Editor ::..

Ciefp Oscam Editor — это плагин для Enigma2, предназначенный для редактирования файлов oscam.dvbapi и oscam.server непосредственно из интерфейса Enigma2.
Он позволяет пользователям легко управлять конфигурацией OSCAM для DVB-API и серверов, отображая информацию о текущем канале и данных ECM.
Плагин предназначен для пользователей, которым требуется устанавливать приоритеты, игнорировать,
сопоставлять или добавлять линии в конфигурации OSCAM, а также управлять считывателями для сервера OSCAM.

Возможности плагина

Отображение информации о канале:
Отображает информацию о текущем просматриваемом канале, включая:
Название канала
ID сервиса
CAID (с выделенными активными CAID)
ECM PID
Провайдер
CHID, считыватель, протокол, количество переходов и время ECM (из /tmp/ecm.info).

Редактирование файла oscam.dvbapi:
Добавление строк: позволяет добавлять различные типы строк:
P: (Приоритет) для установки приоритета CAID и провайдеров.
I: (Игнорировать) для игнорирования определённых CAID или провайдеров.
J: (Объединить) для объединения двух CAID/провайдеров с ECM PID.
M: (Сопоставить) для сопоставления одного провайдера другому.
A: (Добавить) для добавления определённых значений NS, SID и ECM PID.
Возможность добавления комментария с именем канала.
Возможность ограничить строку определённым каналом (ID сервиса).
Просмотр и удаление существующих строк в oscam.dvbapi.

Редактирование файла oscam.server:
Добавление считывателей: настройка новых считывателей с такими параметрами, как:
Метка, протокол (например, CCCam), адрес/порт, пользователь, пароль. Время бездействия, cacheex, группа, кэш EMM, версия CCC и т. д.

Просмотр существующих ридеров на oscam.server.

Удаление выбранных ридеров с подтверждением.

Настройки:
Выберите путь к файлу oscam.dvbapi (например, /etc/tuxbox/config/oscam.dvbapi или варианты для oscam-emu, oscam-master и т. д.).

Выберите язык пользовательского интерфейса (английский или сербский).

Поддержка перевода:
Плагин поддерживает многоязычный интерфейс (в настоящее время английский и сербский) для всех текстов, включая заголовки, подписи к кнопкам и сообщения.

Пользовательский интерфейс:
Интуитивно понятный графический интерфейс с экранами для главного экрана, добавления строк, просмотра файлов, добавления ридеров и настроек.

Поддержка навигации (вверх/вниз) и действий с помощью цветных кнопок (красная, зелёная, жёлтая, синяя).

Заключение
Плагин — полезный инструмент для пользователей OSCAM на устройствах Enigma2, позволяющий легко управлять конфигурацией без необходимости ручного редактирования файлов.
Он подходит для пользователей, которым требуется быстро устанавливать приоритеты, игнорировать или сопоставлять идентификаторы CAID,
а также управлять считывателями, используя дополнительную гибкость многоязычного интерфейса.

osccam-emu - etc/tuxbox/config/osccam-emu/oscam.dvbapi
oscam-master - etc/tuxbox/config/oscam-master/oscam.dvbapi
oscam-smod - etc/tuxbox/config/osccam-smod/oscam.dvbapi
Язык: сербский - английский

..:: Добавить строку dvbapi ::..
Отображение нескольких таблиц ввода текста:
- Приоритет (P:) caid:provider:sid
- Игнорировать (I:) caid:provider:sid
- Присоединить (J:) caid1:provider1:sid:ecmpid caid2:provider2:ecmpid2
- Сопоставить (M:) caid:oldprovider:sid caid2:newprovider:sid2
- Добавить (A:) ns:sid:ecmpid:caid:provider:
Красная кнопка - Отмена
Зелёная кнопка - Добавить - добавляет введённую строку в oscam.dvbapi
Жёлтая кнопка - Предварительный просмотр - открывает экран Предварительный просмотр oscam.dvbapi
