..:: CiefpBouquetUpdater ::..

Что делает плагин?

CiefpBouquetUpdater позволяет легко загружать и обновлять наборы каналов с GitHub (ciefp-E2-75E-34W).
Пользователи могут выбирать, какие наборы каналов установить — спутниковые, тематические или IPTV — без потери текущих настроек.

Как он используется?

1. Запустите плагин из меню ресивера Enigma2.
2. В списке слева выберите нужные наборы (перемещайте стрелки вверх/вниз, подтвердите выбор нажатием «ОК»).
3. Нажмите зелёную кнопку («Копировать») для временного копирования или жёлтую кнопку («Установить») для установки.
4. Дождитесь завершения установки — ресивер автоматически обновится.

В чём особенность?
• Выборочная установка: установите только нужные наборы, а не весь список.
• Сохраняет пользовательские наборы каналов: ваши персональные наборы каналов остаются нетронутыми — старые файлы удаляются только в том случае, если они перекрываются новыми выбранными наборами.
• Отображение имени: вместо технических имён файлов отображаются фактические названия наборов (например, «•• 16.0 E - Max TV ••») в соответствии с порядком, указанным на bouquets.tv.
• Быстро и просто: идеально подходит для всех пользователей, от новичков до продвинутых.

Для кого предназначен?
Этот плагин идеально подходит для тех, кому нужна гибкость в обновлении отдельных наборов каналов, сохраняя при этом свои пользовательские списки на приёмниках Enigma2.
