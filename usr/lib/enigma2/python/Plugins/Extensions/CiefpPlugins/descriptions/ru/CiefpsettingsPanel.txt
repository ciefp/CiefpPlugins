Краткое описание плагина Ciefpsettings Panel
Ciefpsettings Panel — это универсальный плагин для Enigma2, предназначенный для управления и установки различных плагинов,
инструментов, программных камер, скинов и плагинов IPTV на устройства Enigma2.

Предоставляет удобный интерфейс для выбора и установки различных плагинов из онлайн-источников,
облегчая адаптацию и расширение функциональности спутниковых ресиверов.
Плагин особенно полезен для пользователей графических редакторов, таких как OpenPLi, у которых может не быть встроенных менеджеров плагинов.

Основные функции:
1. Централизованная установка плагинов:
o Позволяет выбирать и устанавливать несколько плагинов
o (например, CiefpSettingsDownloader, AjPanel, X-Streamity, программные камеры, такие как Oscam и Ncam, скины, такие как Metrix-FHD) через простой интерфейс.
o Поддержка категоризации плагинов (CiefpSettings, Панели, IPTV, Softcams, Скины, Инструменты) для более удобной навигации.

2. Автоматическая проверка и обновление плагинов:
o При запуске плагин автоматически проверяет наличие новых версий по указанному URL-адресу (VERSION_URL).
o Если доступна новая версия, пользователь может подтвердить обновление,
o при этом список установленных плагинов (ciefp_installed_plugins.txt) сохраняется в резервной копии и восстанавливается после успешного обновления,
что обеспечивает целостность данных.

3. Менеджер плагинов:
o Новая функциональность, позволяющая просматривать и управлять установленными плагинами.
o Пользователи могут удалять выбранные плагины с подтверждением, которое удаляет соответствующие каталоги
o из /usr/lib/enigma2/python/Plugins/Extensions/ и обновляет список установленных плагинов.
o Интерфейс включает строку состояния и кнопки для выбора (зелёная), удаления (красная) и перезапуска Enigma2 (синяя).

4. Установщик IPK:
o Добавлена ​​функция установки файлов .ipk из каталога /tmp, аналогичная функции «Установить локальные расширения» в диспетчере программ.
o Отображает список файлов .ipk в каталоге /tmp, позволяя пользователям выбирать и устанавливать их с помощью команды opkg install.
o Содержит строку состояния для отображения информации об установке, а также кнопки «Выйти» (красная), «Установить» (зелёная) и «Перезапустить Enigma2» (синяя).
o Особенно полезно для образов без встроенного менеджера плагинов, например, OpenPLi.

5. Перезапуск Enigma2:
o Синяя кнопка на всех экранах (Панель настроек Ciefp, Менеджер плагинов, Установщик IPK) позволяет быстро перезапустить Enigma2 с помощью команды init 4 && init 3,
что необходимо после установки или удаления плагина.

Преимущества новых функций:
• Менеджер плагинов упрощает удаление ненужных плагинов без необходимости ручного удаления файлов или использования терминала.
• Установщик IPK расширяет функциональность плагина, позволяя устанавливать локальные файлы .ipk,
• что крайне важно для пользователей, загружающих пакеты из внешних источников.
• Автоматическое обновление гарантирует актуальность плагина, сохраняя пользовательские данные (список установленных плагинов),
• снижая риск потери конфигурации.

Вывод:
Панель Ciefpsettings — мощный инструмент для управления плагинами Enigma2 с новыми функциями, повышающими её практичность и гибкость.
Менеджер плагинов и установщик IPK делают её комплексным решением для установки, удаления и обновления плагинов,
а автоматическое обновление обеспечивает долгосрочную надёжность.
Это идеальное решение для пользователей, которым требуется простое и эффективное управление спутниковыми ресиверами.
