..:: CiefpEPGshare ::..

Плагин Enigma2 для загрузки и отображения данных EPG (электронной программы передач) с сайта epgshare01.online
Позволяет пользователям выбирать XML-файлы с данными EPG, отображать каналы и их программы, а также пиктограммы (иконки каналов).
Плагин оптимизирован для повышения производительности, поддерживает кэширование данных и фильтрацию больших файлов,
и включает в себя настраиваемый пользовательский интерфейс с логотипами для улучшения визуального восприятия.

Выбор XML-файла:
На экране XMLSelectionScreen (разрешение 1280x852) отображается список доступных файлов .xml.gz.
Используйте стрелки вверх и вниз на пульте дистанционного управления для навигации по списку.
Нажмите OK, чтобы выбрать файл и перейти к экрану данных EPG (CiefpEPGshare).

Просмотр данных EPG:
На экране CiefpEPGshare (разрешение 1800x800) отображаются:
Список каналов (слева): список каналов из выбранного XML-файла.
Информация EPG (посередине): программы выбранного канала, отсортированные по дате и времени.
Иконка и логотипы: отображают значок канала (иконку), логотип плагина и фоновые изображения.

Используйте стрелки вверх и вниз для навигации по каналам или данным EPG.
Нажмите «ОК», чтобы переключиться между списком каналов и информацией EPG.
Нажмите «Отмена», чтобы вернуться к предыдущему экрану или выйти.

Игнорирование XML-файлов:
Файл epg_ripper_ALL_SOURCES1.xml.gz автоматически игнорируется из-за его размера
(1,3 ГБ, 22 миллиона строк) во избежание проблем с производительностью.

Чтобы добавить другие XML-файлы в список игнорируемых:
На экране выбора XML выберите файл, который хотите игнорировать.
Нажмите красную кнопку на пульте дистанционного управления. Файл будет добавлен в /tmp/CiefpEPGshare/ignore_list.txt и исключён из отображения.
Игнорируемые файлы не отображаются в списке, пока вы вручную не удалите их из ignore_list.txt (например, через SSH или FTP).

Логи и диагностика:
При возникновении ошибок (например, проблем с сетью или загрузкой логотипа) проверьте файл /tmp/ciefp_epgshare.log для получения подробной информации.
Плагин автоматически пытается установить библиотеку lxml, если она отсутствует, или использует ElementTree в качестве замены.
