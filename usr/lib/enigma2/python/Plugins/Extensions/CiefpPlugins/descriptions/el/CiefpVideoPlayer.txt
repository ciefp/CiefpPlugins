# CiefpVideoPlayer v1.3

**Το CiefpVideoPlayer** είναι ένα προηγμένο πρόγραμμα αναπαραγωγής βίντεο για συσκευές Enigma2 με πλήρη έλεγχο αναπαραγωγής.

## 🎬 Βασικά Χαρακτηριστικά

### 📺 Προηγμένο Πρόγραμμα Αναπαραγωγής Βίντεο
- **Πλήρης Έλεγχος Αναπαραγωγής** – Αναπαραγωγή, Παύση, Διακοπή, Πίσω, Εμπρός
- **Απομνημόνευση Θέσης** – συνέχιση της παρακολούθησης από το σημείο που την αφήσατε
- **Ροές HTTP** – παύση και συνέχιση διαδικτυακού περιεχομένου
- **Υπότιτλοι** – υποστήριξη για ενσωματωμένους και εξωτερικούς υπότιτλους

### 📁 Υποστηριζόμενες Πηγές
- **Τοπικά Αρχεία** – HDD, USB, Κάρτες CF
- **Κοινόχρηστα Δίκτυα** – SMB/CIFS, NFS
- **Διαδικτυακές Ροές** – Λίστες Αναπαραγωγής M3U και Enigma2 .tv
- **Ανοιχτός Κατάλογος** – περιήγηση και αναπαραγωγή ταινιών/σειρών από διακομιστές HTTP

### 🎭 Ενσωμάτωση TMDB
- **Ταινίες** – λεπτομερείς πληροφορίες, αξιολογήσεις, ηθοποιοί (κίτρινο κουμπί)
- **Σειρές** – λεπτομέρειες για σεζόν, επεισόδια, ηθοποιούς (μπλε κουμπί)
- **Αφίσες** – αυτόματη προσωρινή αποθήκευση για ταχύτερη προβολή
- **Αξιολογήσεις IMDb** – μέσω OMDb API (προαιρετικό)

### 💾 Αποθήκευση στην προσωρινή μνήμη
- Οι αφίσες και τα μεταδεδομένα αποθηκεύονται στην προσωρινή μνήμη για ταχύτερη φόρτωση
- Ρυθμιζόμενο μέγεθος προσωρινής μνήμης
- Επιλογή αυτόματης διαγραφής κατά την έξοδο

| Κουμπί | Λειτουργία |

| **Αναπαραγωγή/Παύση** | Παύση και συνέχιση |

| **◀ ▶** | Επαναφορά/Γρήγορη προώθηση |

| **Έξοδος** | Έξοδος από το πρόγραμμα αναπαραγωγής (θυμάται τη θέση!) |

## ⏯️ Συνέχιση αναπαραγωγής

Το MoviePlayer θυμάται αυτόματα τη θέση από την οποία σταματήσατε. Την επόμενη φορά που θα αναπαράγετε το ίδιο βίντεο, το Enigma2 θα σας ρωτήσει αν θέλετε να συνεχίσετε από εκεί που σταματήσατε.

**Υποστηρίζεται για:**
- ✅ Τοπικά αρχεία (HDD, USB)
- ✅ Κοινόχρηστα στοιχεία δικτύου (SMB/NFS)
- ✅ Ροές HTTP (Ανοιχτός Κατάλογος)
- ✅ Λίστες αναπαραγωγής M3U και .tv

## 🚀 Πώς λειτουργεί

**Παλιός τρόπος (v1.2 και παλαιότερες):**

ref = eServiceReference(4097, 0, path)
session.nav.playService(ref) # Χωρίς έλεγχο, χωρίς συνέχιση

Νέος τρόπος (v1.3):

ref = eServiceReference(4097, 0, path)
ref.setName(name)
session.open(MoviePlayer, ref) # Πλήρης έλεγχος, υποστήριξη συνέχισης!
