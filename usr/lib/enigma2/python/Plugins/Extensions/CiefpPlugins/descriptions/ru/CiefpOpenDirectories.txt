..:: CiefpOpenDirectories ::..

Плагин Enigma2 для просмотра открытых каталогов (Open Directories) в интернете.

### Обеспечивает: ####
Загрузку URL-адресов из файла opendirectories.txt.
Навигацию по папкам и выбор медиафайлов (видео: MP4, MKV, AVI, TS; аудио: MP3, FLAC).
Создание плейлистов M3U для Media Player.
Создание пользовательских наборов Enigma2 с корректными типами SERVICE (4097:0:1 для видео, 4097:0:2 для аудио с постерами радио).
Поддержка HTTPS/HTTP, защита от тайм-аута и автоматическая перезагрузка набора.

Идеально подходит для быстрого создания миксов IPTV из контента открытых каталогов. Версия 1.0 — просто, быстро и надежно!

### Как использовать ###
- Запустите плагин, выберите адрес
- Зелёная кнопка или кнопка «ОК» для навигации по папкам
- Навигация вверх и вниз с помощью стрелок
- Выбор файла кнопкой «ОК»
- Жёлтая кнопка для создания файла m3u или userbouquet
- Синяя кнопка для выбора всех файлов в папке

Из одного адреса можно объединить несколько папок
в один файл m3u или userbouquet
После завершения добавления файлов
Жёлтая кнопка для создания файла m3u или userbouquet
При нажатии на жёлтую кнопку появляется виртуальная клавиатура
Здесь вы можете изменить имя или
оставить сгенерированное имя файла
При нажатии на зелёную кнопку создаётся выбранный файл

Файлы m3u сохраняются в /tmp/CiefpOpenDirectories
Userbuket устанавливаются в etc/enigma2, перезагрузка не требуется
