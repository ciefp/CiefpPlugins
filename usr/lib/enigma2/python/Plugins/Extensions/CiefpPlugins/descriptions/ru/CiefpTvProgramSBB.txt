..:: CiefpTvProgramSBB ::..

Плагин Enigma2 для отображения телепрограмм (EPG) для SBB Сербии.
Загружает данные EPG с сайта epgshare01.online в формате XML,
кэширует их локально и отображает в пользовательском интерфейсе с каналами,
программами и пиктограммами (логотипами каналов).

Поддерживает навигацию между списком каналов и информацией EPG с автоматическим позиционированием на текущей программе.

Использование:
Установка: Плагин устанавливается в среде Enigma2. Требуется модуль lxml; если он отсутствует, плагин попытается установить его автоматически.
Запуск: Доступен через меню плагинов. Открывается экран со списком каналов и данными EPG.

Навигация:
Список каналов: Отображает доступные каналы. Навигация осуществляется клавишами «вверх»/«вниз».
Вид EPG: Отображает программы выбранного канала, упорядоченные по дате, времени, названию, категории и описанию.
Переключение фокуса: клавиша OK переключает фокус между списком каналов и информацией EPG.
Выход: кнопка «Отмена» закрывает плагин.
Пиктограммы: автоматически загружает логотипы каналов из каталога пиктограмм или использует файл placeholder.png по умолчанию, если пиктограммы недоступны.
Кэширование: данные EPG кэшируются в /tmp/CiefpTvProgramSBB на 24 часа для более быстрой загрузки.
Примечание: плагин требует стабильного подключения к Интернету для загрузки данных EPG и записывает ошибки в /tmp/ciefp_epgshare.log.
