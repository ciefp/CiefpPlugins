- Плагин музыкального плеера Enigma2
### Описание:
CiefpVibes — отличный плагин музыкального плеера для Enigma2.
Воспроизведение локальных или онлайн-плейлистов (.m3u/.tv) с повтором, 
случайным воспроизведением, настраиваемыми фонами и интеграцией с GitHub для получения новых треков.

Поддерживаемые форматы: mp3 и flac.
Идеально для непрерывного звучания!

### Использование:
Запуск из меню плагинов.

Зелёный: Загрузка локального файла.

Синий: Подключение к GitHub.
Жёлтый: Настройки.
Красный: Выход.
Навигация по плейлисту, нажмите кнопку «ОК» для воспроизведения.
Автоматическое сохранение последнего сеанса.

Почувствуйте позитив с плагином CiefpVibes для enigma2
