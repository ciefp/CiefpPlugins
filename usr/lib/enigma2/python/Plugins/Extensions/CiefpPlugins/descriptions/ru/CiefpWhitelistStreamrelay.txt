..:: CiefpWhitelistStreamrelay ::..

CiefpWhitelistStreamrelay — плагин для приёмников Enigma2, предназначенный для автоматизированной генерации
файла `whitelist_streamrelay` с использованием данных из предопределённых файлов userbouquet.

Основные функции:
1. Генерация файла `whitelist_streamrelay`:
- Плагин обрабатывает данные из выбранных файлов `userbouquet` и извлекает допустимые строки, содержащие
информацию о потоках.
- Создаёт или обновляет файл `whitelist_streamrelay` в каталоге `/etc/enigma2`.
- Поддерживаемые пакеты: SKYDE 19.0E - SKYUK 28.2E.

2. Отображение информации для пользователя:
- Отображает количество строк, добавленных в файл `whitelist_streamrelay`, предоставляя пользователю обзор результатов обработки.

3. Перезапуск Enigma2:
- После завершения генерации файла пользователь может подтвердить перезапуск Enigma2 для загрузки новых настроек.
- Перезапуск выполняется только после нажатия кнопки «ОК» на пульте дистанционного управления.

Зачем использовать этот плагин?
- Он подходит для пользователей, которые часто обновляют свои списки ретрансляторов и хотят автоматизировать процесс без ручного редактирования файлов.
- Простой и интуитивно понятный интерфейс позволяет просматривать сгенерированные строки и управлять перезапуском системы.
- Этот плагин — практичный инструмент для продвинутых пользователей системы Enigma2, которым требуется более эффективное управление и организация конфигураций ретрансляторов.
Этот плагин был создан специально для моих настроек, с другими настройками он работать не будет.
