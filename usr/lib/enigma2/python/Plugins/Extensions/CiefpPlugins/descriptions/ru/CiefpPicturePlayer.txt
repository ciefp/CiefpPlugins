🎮 Кнопки (главный экран)
Функции кнопок
◀ ▲ ▼ ▶ Прокрутка списка
OK Открыть папку / отобразить изображение в полноэкранном режиме
ВЫХОД (КРАСНЫЙ) Выйти из плагина
ПАПКА (ЗЕЛЕНЫЙ) Открыть браузер папок (локальные файлы)
СЕТЬ (ЖЕЛТЫЙ) Подключиться к сетевым ресурсам (SMB)
ОНЛАЙН (СИНИЙ) Загрузить изображения из GitHub
МЕНЮ Открыть настройки (очистка кэша, информация)

🖼️ Отображение изображений
Предварительный просмотр – изображение отображается справа при прокрутке списка
Полноэкранный режим - нажмите OK на изображении
Слайд-шоу – в полноэкранном режиме нажмите ЗЕЛЕНУЮ кнопку
Информация об изображении – в полноэкранном режиме нажмите СИНЮЮ кнопку
Назад – в полноэкранном режиме нажмите КРАСНУЮ кнопку

🌐 Сетевое подключение (SMB)
Нажмите ЖЕЛТУЮ кнопку (СЕТЬ) → Подключиться Подключение к ноутбуку (SMB)
Введите IP-адрес ноутбука (например, 192.168.1.100)
Введите имя общей папки (или оставьте поле пустым)
После успешного подключения вы будете перенаправлены в папку с изображениями
#Автоматическое сканирование
- выберите IP-адрес устройства
- введите имя общей папки

📡 Онлайн-контент
Нажмите СИНИЙ (ОНЛАЙН)
Выберите файл .tv из списка
Плагин загрузит и отобразит изображения из этого файла

⚙️ Настройки (кнопка МЕНЮ)
Функции параметров
Очистка кэша изображений Удаляет кэшированные изображения (освобождает место в /tmp)
Информация о кэше Отображает размер и количество файлов в кэше
Язык / Язык Изменить язык (подготовлено для будущих версий)
Тема / Тема Изменить тему (подготовлено для будущих версий)

⚠️ Известные проблемы
Проблема Решение
Сетевое монтирование не работает в OpenPLi Установите cifs-utils: opkg install cifs-utils
Медленная загрузка больших изображений Изображения масштабируются автоматически, подождите несколько секунд
.JPG не отображается Уже исправлено в версии 1.0 – используется os.path.splitext
