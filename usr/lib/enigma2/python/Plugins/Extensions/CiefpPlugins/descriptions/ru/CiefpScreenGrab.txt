====================================
Плагин CiefpScreenGrab — руководство пользователя
=====================================
Версия: 1.2
CiefpScreenGrab — это плагин для Enigma2 для захвата скриншотов на вашем
спутниковом ресивере. Он поддерживает удалённое управление скриншотами,
минималистичный интерфейс, просмотр и удаление сохранённых изображений, а также настройку формата и разрешения.

Конфигурация
=============
1. Откройте плагин из меню: Меню -> Плагины -> CiefpScreenGrab.
2. Настройки:
- Включить CiefpScreenGrab: включить плагин (по умолчанию: True).
- Использовать минималистичный скин: включить минималистичный интерфейс (маленькое окно
в правом верхнем углу, по умолчанию: True).
- Снимок экрана: выберите, что нужно захватить:
* OSD + Видео (по умолчанию)
* Только видео
* Только OSD
- Формат скриншотов: JPG или PNG (по умолчанию: PNG).
* Если JPG, настройте качество JPG-изображения (10–100, по умолчанию: 100).
- Размер изображения (ширина): разрешение изображения (по умолчанию: разрешение скина,
варианты: 720, 1280, 1920).
- Путь к скриншотам: место сохранения изображений (по умолчанию:
/media/hdd/CiefpScreenGrab, параметры: /media/hdd, /media/usb, /tmp).
– Выберите кнопку для создания снимка экрана: кнопка на пульте дистанционного управления для захвата
(по умолчанию: нет).
* Варианты: отключение звука, справка, информация, таймер, EPG, ТВ, радио, текст, аудио,
красный, зеленый, желтый, синий.
* Для красного, зеленого и желтого: работает только долгое нажатие.
* Для других: выберите короткое или долгое нажатие (настройте
используйте кнопку «[button]» вместо «[button] long»).
– Время ожидания информационного сообщения: длительность уведомления (по умолчанию: 3
секунды, варианты: 1, 3, 5, 10, выкл., без тайм-аута).
3. Сохранить: зеленая кнопка.
4. Сбросить к настройкам по умолчанию: желтая кнопка (требуется подтверждение).

[Примечание]: убедитесь, что выбранное устройство хранения (например, /media/hdd)
доступно и доступно для записи. Если нет, измените путь.

Использование
=====
Снимок экрана
----------------------
1. С помощью кнопки:
- Выберите кнопку в настройках (например, синюю).
- Нажмите выбранную кнопку (короткое или длинное нажатие, в зависимости от
настроек).
- Скриншот будет сделан и отображен немедленно.
2. С помощью Minimal Skin:
- Откройте плагин (если включен параметр «Использовать минимальный скин»).
- Нажмите зелёную кнопку (Снимок экрана).
- Скриншот будет сделан и отображен.
3. Предварительный просмотр после съёмки:
- После съёмки откроется экран предварительного просмотра с изображением (1250x750).
- Варианты:
* Красная кнопка (Выход): Закрывает предварительный просмотр.
* Жёлтая кнопка (Файлы): Открывает список сохранённых изображений.

Просмотр и удаление изображений
--------------------------
1. Открыть список изображений:
- Из Minimal Skin: Жёлтая кнопка (Файлы).
– В настройках: Синяя кнопка (Файлы).
– В режиме предварительного просмотра скриншота: Жёлтая кнопка (Файлы).
2. В FileScreen:
– Слева: Список файлов (самые новые вверху).
– Справа: Предварительный просмотр выбранного изображения (1300x700).
– Кнопка OK: Выбор/отмена выбора файла (отмеченного [X]).
– Зелёная кнопка (Выбрать все): Выбор всех файлов.
– Красная кнопка (Удалить): Удаление выбранных файлов (требуется
подтверждение).
– Отмена/Выход: Закрытие экрана.

Минимальный скин
------------
– Если включен параметр «Использовать минимальный скин», плагин открывается в виде небольшого окна
(540x40) в правом верхнем углу.
– Функции:
* Красный (Выход): Закрывает окно.
* Зелёный (Снимок экрана): Снимок экрана.
* Жёлтый (Файлы): Открывает список изображений.
* Синяя кнопка (Настройки): Открывает настройки.

Рекомендуемые настройки
=====================
- Формат: PNG для наилучшего качества без потерь.
- Разрешение: разрешение скина или 1280 для баланса качества и
производительности.
- Путь: /media/hdd для постоянного хранения.
- Кнопка: Синяя или Красная (длительное нажатие) для удобства захвата.
- Время ожидания: 3 секунды для уведомлений вместе с предварительным просмотром изображения.

Примечания
=====
- Плагин протестирован на ресиверах Vu+ (например, Zero 4K) с
изображениями OpenPLi и OpenATV.
- Изображения сохраняются в /media/[hdd/usb/tmp]/CiefpScreenGrab/.
- Избегайте использования кнопок, уже назначенных в системе (например, EPG, Информация).

Приятного вам захвата скриншотов с помощью CiefpScreenGrab!
