..:: CiefpTMDBSearch v1.0 ::..

Αναζητήστε ταινίες και σειρές μέσω του TMDB με αφίσες, περιγραφές, ηθοποιούς και αξιολογήσεις – σε 14 γλώσσες!
Το μόνο που χρειάζεστε είναι ένα δωρεάν κλειδί API TMDB (εισάγετε μία φορά μέσω του κίτρινου κουμπιού).

# ΒΑΣΙΚΕΣ ΛΕΙΤΟΥΡΓΙΕΣ #
Αναζήτηση ταινιών (ΠΡΑΣΙΝΟ) και σειρών (ΚΙΤΡΙΝΟ)
Εμφάνιση λεπτομερειών: τίτλος, έτος, διάρκεια, βαθμολογία, είδος, σκηνοθέτης/δημιουργός, ηθοποιοί, περιγραφή
Λήψη αφισών (εξώφυλλα) από το TMDB
Υποστήριξη για πολλές γλώσσες (περιγραφή, τίτλοι)
Αποθήκευση αφισών στην προσωρινή μνήμη (ταχύτερη πρόσβαση, λιγότερη κίνηση στο διαδίκτυο)
Χειροκίνητη ρύθμιση κλειδιού API TMDB

# ΕΓΚΑΤΑΣΤΑΣΗ ΚΑΙ ΡΥΘΜΙΣΗ #

1. Κλειδί API (ΥΠΟΧΡΕΩΤΙΚΟ)
Εγγραφείτε στο themoviedb.org και λάβετε ένα κλειδί API
Στο πρόσθετο, πατήστε ΜΠΛΕ → Ρυθμίσεις → ΚΙΤΡΙΝΟ (Κλειδί API)
Εισαγάγετε το κλειδί API σας μέσω του εικονικού πληκτρολογίου

2. Το κλειδί API αποθηκεύεται στο tmdbapikey.txt
Μπορείτε να γράψετε το κλειδί στο αρχείο κειμένου tmdbapikey.txt
και να το μετακινήσετε στον φάκελο του πρόσθετου
όταν ξεκινήσετε, το πρόσθετο θα το διαβάσει.

# ΧΡΗΣΗ #
Κύριο μενού:
ΠΡΑΣΙΝΟ → Αναζήτηση ταινιών
ΚΙΤΡΙΝΟ → Αναζήτηση σειρών
ΜΠΛΕ → Ρυθμίσεις
ΚΟΚΚΙΝΟ → Έξοδος

Παράδειγμα αναζήτησης:
Πατήστε ΠΡΑΣΙΝΟ για ταινίες
Εισαγάγετε το όνομα της ταινίας (π.χ. "Inception")
Το πρόσθετο εμφανίζει:
Τίτλος, έτος, διάρκεια
Βαθμολογία (π.χ. 8.3/10 ★)
Είδος, σκηνοθέτης
Σύντομη περιγραφή πλοκής
Κύριοι ηθοποιοί
Αφίσα (αν διατίθεται)

# ΡΥΘΜΙΣΕΙΣ (ΜΠΛΕ) #
Κλειδί API (ΚΙΤΡΙΝΟ) - ορίστε το κλειδί σας
Γλώσσα (ΜΠΛΕ) - αλλαγή γλώσσας
Λήψη αφισών - ενεργοποίηση/απενεργοποίηση λήψης αφισών
Εκκαθάριση προσωρινής μνήμης (ΜΕΝΟΥ) - διαγραφή όλων των αποθηκευμένων αφισών
Αποθήκευση (ΠΡΑΣΙΝΟ) - αποθήκευση αλλαγών

# ΠΡΟΣΩΡΙΝΗ ΜΝΗΜΗ #
Οι αφίσες αποθηκεύονται αυτόματα στο /tmp/CiefpTMDBSearch/
Εξοικονομεί κίνηση στο διαδίκτυο κατά την εκ νέου αναζήτηση
Μπορείτε να την απενεργοποιήσετε στις ρυθμίσεις, αν θέλετε

# ΥΠΟΣΤΗΡΙΖΟΜΕΝΕΣ ΓΛΩΣΣΕΣ #
Αγγλικά, Σερβικά, Κροατικά, Βοσνιακά, Σλοβακικά, Γερμανικά, Ισπανικά,
Γαλλικά, Ιταλικά, Ρωσικά, Πορτογαλικά, Πολωνικά, Τουρκικά, Αραβικά, Κινέζικα.

Δοκιμασμένο και λειτουργεί άψογα σε:
OpenATV · OpenBH · OpenPLI · Pure2 · Egami · OpenVision
