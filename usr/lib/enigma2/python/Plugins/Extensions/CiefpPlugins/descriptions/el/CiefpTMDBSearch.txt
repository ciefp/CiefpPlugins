Αναζητήστε ταινίες και σειρές μέσω του TMDB με αφίσες, περιγραφές, ηθοποιούς και αξιολογήσεις - σε 14 γλώσσες!
Απαιτείται μόνο ένα δωρεάν κλειδί API TMDB (που εισάγεται μία φορά μέσω του κίτρινου κουμπιού).

# ΒΑΣΙΚΕΣ ΛΕΙΤΟΥΡΓΙΕΣ #
Αναζήτηση ταινιών (ΠΡΑΣΙΝΟ) και σειρών (ΚΙΤΡΙΝΟ)
Λεπτομέρειες εμφάνισης: τίτλος, έτος, διάρκεια, βαθμολογία, είδος, σκηνοθέτης/δημιουργός, ηθοποιοί, περιγραφή
Λήψη της αφίσας (εξώφυλλο) από το TMDB
Υποστήριξη πολλαπλών γλωσσών (περιγραφή, τίτλοι)
Αποθήκευση αφίσας στην προσωρινή μνήμη (ταχύτερη πρόσβαση, λιγότερη κίνηση στο διαδίκτυο)
Χειροκίνητη ρύθμιση του κλειδιού API του TMDB

# ΕΓΚΑΤΑΣΤΑΣΗ ΚΑΙ ΡΥΘΜΙΣΗ #
1. Κλειδί API (ΥΠΟΧΡΕΩΤΙΚΟ)
Εγγραφείτε στο themoviedb.org και λάβετε ένα κλειδί API
Στο πρόσθετο, πατήστε ΜΠΛΕ → Ρυθμίσεις → ΚΙΤΡΙΝΟ (Κλειδί API)
Εισαγάγετε το κλειδί API σας μέσω του εικονικού πληκτρολογίου

2. Το κλειδί API αποθηκεύεται στο tmdbapikey.txt
Μπορείτε να εισάγετε το κλειδί στο αρχείο κειμένου tmdbapikey.txt
και να το μεταφέρετε στον φάκελο του πρόσθετου
Κατά την εκκίνηση του πρόσθετου θα το διαβάσει.

Τέσσερις νέες επιλογές έχουν προστεθεί στο μενού Σύνθετη αναζήτηση:

Αναζήτηση δημοφιλών ταινιών
– Εμφανίζει τις 20 πιο δημοφιλείς ταινίες που υπάρχουν αυτήν τη στιγμή στο TMDB (μακροπρόθεσμη δημοτικότητα).
– Η λίστα περιέχει: όνομα ταινίας, έτος κυκλοφορίας, βαθμολογία TMDB και αστέρια.
- Κάνοντας κλικ στην ταινία ανοίγει μια πλήρης προβολή με την αφίσα, την περιγραφή, τους ηθοποιούς κ.λπ.

Αναζήτηση δημοφιλών σειρών
– Εμφανίζει τις 20 πιο δημοφιλείς σειρές που υπάρχουν αυτήν τη στιγμή στο TMDB.
– Εμφάνιση: όνομα σειράς, έτος έναρξης, βαθμολογία και αστέρια.
– Η επιλογή ανοίγει μια λεπτομερή προβολή της σειράς.
Αναζήτηση δημοφιλών προσώπων
– Εμφανίζει τα 20 πιο δημοφιλή πρόσωπα (ηθοποιοί, σκηνοθέτες κ.λπ.).
– Εμφάνιση: όνομα ατόμου, τομέας (π.χ. Υποκριτική/Σκηνοθεσία), σύντομη λίστα διάσημων ταινιών/σειρών, δημοτικότητα με τους σταρ.
- Κάντε κλικ για να ανοίξετε το προφίλ του ατόμου με βιογραφία, φιλμογραφία και αφίσα.
Αναζήτηση σε δημοφιλή Όλα (Καθημερινά)
– Εμφανίζει τους 20 πιο πρόσφατους τίτλους και πρόσωπα ανά ημέρα (μικτά – ταινίες, σειρές και πρόσωπα),
όπως ακριβώς και η ενότητα "Τάσεις" στην αρχική σελίδα του TMDB.
– Ενημερώνεται καθημερινά και δείχνει τι είναι πρόσφατο o τα πιο σημαντικά (νέες επιτυχίες, viral σειρές, τρέχοντες ηθοποιοί).
– Εμφάνιση:
• Ταινίες: [Ταινία] Τίτλος (έτος) ★★★★★ 7.8
• Σειρά: [Σειρά] Τίτλος (έτος) ★★★★★ 8.5
• Πρόσωπα: [Άτομο] Όνομα Επώνυμο • Υποκριτική ★★★★ 45.2
– Η επιλογή ανοίγει την αντίστοιχη λεπτομερή προβολή (προφίλ ταινίας/σειράς/ατόμου).

Τα υπόλοιπα

Όλες οι εμφανίσεις περιορίζονται σε 20 αποτελέσματα για ταχύτητα και σαφήνεια (όπως στον ιστότοπο TMDB - πρώτη σελίδα).
Οι αξιολογήσεις εμφανίζονται με αστέρια για ταχύτερη προβολή.
Υποστηρίζεται η επιλογή γλώσσας περιγραφής (μέσω ρυθμίσεων plugin).

Το πρόσθετο πλέον διαθέτει:
✅ Αυτόματη αναζήτηση EPG - αυτόματη ανίχνευση του τρέχοντος προγράμματος
✅ Σύνθετη αναζήτηση - ταινίες, σειρές, ηθοποιοί, σκηνοθέτες
✅ Λίστες δημοφιλών/προτεινόμενων - τι είναι δημοφιλές αυτήν τη στιγμή
✅ Εξερεύνηση ηθοποιών - εξερεύνηση ηθοποιών
✅ Σεζόν και επεισόδια - λεπτομερής επισκόπηση τηλεοπτικών σειρών και επεισοδίων
✅ Βαθμολογίες TMDB και IMDB - πρόσθετες πληροφορίες
✅ Συλλογή Backdrop - προεπισκόπηση εικόνας φόντου
✅ Συλλογή αφισών - προεπισκόπηση αφίσας
✅ Ρυθμίσεις - Κλειδιά API, γλώσσα, προσωρινή μνήμη

🖼️ Σύντομος οδηγός: Πώς να χρησιμοποιήσετε το πρόγραμμα προβολής αφισών και φόντου
Η νέα συλλογή είναι εύχρηστη. Ακολουθεί ένας σύντομος οδηγός:

Πρόσβαση στη συλλογή:
Αφού φορτώσετε τις πληροφορίες της ταινίας ή της σειράς, πατήστε το κουμπί MENU στο τηλεχειριστήριό σας.
Στο μενού, επιλέξτε την επιλογή 🖼️ Συλλογή φόντου ή 🎬 Συλλογή αφισών.

Πλοήγηση στη συλλογή:
Χρησιμοποιήστε τα ΚΙΤΡΙΝΑ και ΜΠΛΕ κουμπιά (ή τα αριστερά/δεξιά βέλη) για να πλοηγηθείτε στις εικόνες.

Το ΠΡΑΣΙΝΟ κουμπί σας επιστρέφει στην αρχική οθόνη ή εναλλάσσεται μεταξύ της συλλογής φόντου και αφισών (αν είναι δυνατόν).
Η γραμμή κατάστασης στο επάνω μέρος δείχνει ποια εικόνα είστε από το σύνολο του πόσου (π.χ., "6/185").

Κλείσιμο της συλλογής:
Πατήστε το ΚΟΚΚΙΝΟ κουμπί ή OK/Έξοδος για να επιστρέψετε στην κύρια οθόνη του πρόσθετου.

💡 Συμβουλές και σημειώσεις
Αποθήκευση στην προσωρινή μνήμη: Οι εικόνες λαμβάνονται και αποθηκεύονται αυτόματα στον επιλεγμένο φάκελο προσωρινής μνήμης (ορίζεται στις Ρυθμίσεις). Αυτό επιταχύνει την προβολή την επόμενη φορά που θα την ανοίξετε.
Ρύθμιση της γλώσσας: Στις Ρυθμίσεις, μπορείτε να αλλάξετε τη γλώσσα για τις περιγραφές. Αυτό επηρεάζει επίσης τη γλώσσα των δεδομένων που λαμβάνετε από τη βάση δεδομένων TMDB.
Επιδόσεις: Εάν το πρόσθετο αρχίσει να λειτουργεί πιο αργά λόγω πλήρους προσωρινής μνήμης, μπορείτε να το διαγράψετε μέσω του ΜΕΝΟΥ > Περισσότερες επιλογές > 🗑️ Διαγραφή προσωρινής μνήμης.

Η νέα συλλογή είναι μια εξαιρετική προσθήκη που παρέχει στους χρήστες μια πλήρη εμπειρία προβολής του οπτικού περιεχομένου μιας ταινίας ή σειράς.
