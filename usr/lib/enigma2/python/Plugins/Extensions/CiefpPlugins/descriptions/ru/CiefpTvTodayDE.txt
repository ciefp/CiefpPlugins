..:: CiefpTvTodayDE ::..

CiefpTvTodayDE — это плагин Enigma2 для отображения данных EPG (электронного программного гида) телеканалов,
адаптированный для пользователей в Германии. Плагин загружает данные EPG с сайта epgshare01.online,
кэширует их локально и отображает программы каждого канала вместе с соответствующими пиктограммами (логотипами каналов).
Интерфейс позволяет легко просматривать каналы и их программы, выделяя текущую или следующую программу.

Отображение списка каналов и данных EPG (время, название, категория, описание).
Автоматическая загрузка и кэширование данных EPG (24-часовой кэш).
Отображение пиктограмм каналов с возможностью возврата к пиктограмме-заполнителю, если пиктограмма отсутствует.
Интуитивно понятный интерфейс с панелью выбора, которая автоматически отмечает текущую или следующую программу.
Поддержка навигации с помощью клавиш OK, UP, DOWN и EXIT.
В файл /tmp/ciefp_epgshare.log записываются только критические ошибки и отсутствующие пиктограммы.

Навигация по интерфейсу:
Список каналов: расположен в левой части экрана. Используйте клавиши «ВВЕРХ» и «ВНИЗ» для прокрутки списка каналов.
Данные EPG: отображаются в правой части экрана для выбранного канала, включая время, название, категорию и описание программы.

Панель выбора: автоматически выделяет текущую программу (если доступна) или следующую программу на выбранный день.

Если программ на текущий день нет, выделяется последняя доступная программа.

Значки и логотипы: значок выбранного канала отображается в нижней части экрана (слева). Если значок отсутствует, используется placeholder.png.

Элементы управления:
OK: переключает фокус между списком каналов и данными EPG. При фокусе на каналах выбирает канал;
при фокусе на данных EPG позволяет прокручивать программы. ВВЕРХ/ВНИЗ: Перемещение курсора по списку каналов (когда фокус находится на каналах) или по программам EPG (когда фокус находится на данных EPG).
ВЫХОД: Закрывает плагин и возвращает на предыдущий экран.
