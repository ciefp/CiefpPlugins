..:: CiefpTMDBSearch v1.0 ::..

Поиск фильмов и сериалов через TMDB по постерам, описаниям, актёрам и рейтингам — на 14 языках!
Всё, что вам нужно, — это бесплатный ключ API TMDB (введите его один раз, нажав жёлтую кнопку).

# ОСНОВНЫЕ ФУНКЦИИ #
Поиск фильмов (ЗЕЛЕНЫЙ) и сериалов (ЖЁЛТЫЙ)
Показать подробности: название, год, продолжительность, рейтинг, жанр, режиссёр/создатель, актёры, описание
Скачивание постеров (обложек) из TMDB
Поддержка нескольких языков (описание, названия)
Кэширование постеров (более быстрый доступ, меньше трафика)
Ручная установка ключа API TMDB

# УСТАНОВКА И НАСТРОЙКА #

1. Ключ API (ОБЯЗАТЕЛЬНО)
Зарегистрируйтесь на themoviedb.org и получите ключ API
В плагине нажмите СИНЮЮ кнопку → Настройки → ЖЁЛТЫЙ (Ключ API)
Введите свой ключ API с помощью виртуальной клавиатуры

2. Ключ API хранится в файле tmdbapikey.txt
Вы можете записать ключ в текстовый файл tmdbapikey.txt
и переместить его в папку плагина
плагин считывает его при запуске.

# ИСПОЛЬЗОВАНИЕ #
Главное меню:
ЗЕЛЁНЫЙ → Поиск фильмов
ЖЁЛТЫЙ → Поиск сериалов
СИНИЙ → Настройки
КРАСНЫЙ → Выход

Пример поиска:
Нажмите ЗЕЛЁНУЮ кнопку для поиска фильмов
Введите название фильма (например, «Начало»)
Плагин отображает:
Название, год, продолжительность
Рейтинг (например, 8,3/10 ★)
Жанр, режиссёр
Краткое описание сюжета
Исполнители главных ролей
Постер (если доступен)

# НАСТРОЙКИ (СИНИЙ) #
Ключ API (ЖЁЛТЫЙ) - установите свой ключ
Язык (СИНИЙ) - измените язык
Загрузить постеры - включите/отключите загрузку постеров
Очистить кэш (МЕНЮ) - удалите все сохранённые постеры
Сохранить (ЗЕЛЁНЫЙ) - сохраните изменения

# КЭШ #
Постеры автоматически сохраняются в /tmp/CiefpTMDBSearch/
Экономит интернет-трафик при повторном поиске
Вы можете При желании отключите эту функцию в настройках.
