Плагин - Описание и использование
Что делает плагин?
- Позволяет выбирать спутники из списка доступных.
- Фильтрует тематические пакеты (EX-YU, Спорт, Кино, Музыка, Документальные фильмы и т.д.) в соответствии с выбранными спутниками.
- Автоматически распознаёт маркеры спутников (#SERVICE 1:64) и добавляет все каналы (#SERVICE 1:0), которые им принадлежат.
- Выполняет поиск в пакетах и ​​копирует только те каналы, которые принадлежат выбранным спутникам.
- Спутниковые пакеты копируются на заданные карты и в соответствии с выбранными спутниками.
- Пакеты IPTV, Радио и DVB.T/T2 автоматически копируются с каждым списком.
- Устанавливает отфильтрованные пакеты в систему Enigma2 и позволяет пользователям смотреть только нужные спутники.
- Предоставляет пользователю возможность создать собственный список в соответствии с его потребностями.

Как использовать плагин?
Запуск плагина

Установите и запустите CiefpSelectSatellite из меню плагина Enigma2.
Плагин загрузит список доступных спутников из satellites.xml.

# Выбор спутников
Используйте стрелки на пульте дистанционного управления, чтобы выбрать нужные спутники.
Нажмите OK, чтобы добавить или удалить спутник из списка.
Выбранные спутники появятся в правой части экрана.

# Копирование набора спутников
Нажмите ЗЕЛЁНУЮ кнопку, чтобы отфильтровать наборы спутников и каналы по выбранным спутникам.
Плагин выполнит поиск по наборам спутников, сохранит только каналы с выбранных спутников и удалит остальные.

# Установка набора спутников
Нажмите ЖЁЛТУЮ кнопку, чтобы установить отфильтрованные наборы спутников в /etc/enigma2/.
Плагин автоматически обновит Enigma2 и активирует новые настройки.

# Закрытие
Нажмите КРАСНУЮ кнопку (СТРОКА), чтобы выйти из плагина.

# Примечание
- Плагин всегда загружает последние доступные настройки мотора с GitHub.
- Для тех, кому недостаточно имеющихся версий, теперь есть возможность создать свою собственную.
- Результат: Теперь пользователь может создавать настройки в соответствии со своими потребностями!
